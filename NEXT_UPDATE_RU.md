# Fomo Flight - следующее обновление

## P0: починить Telegram Stories

Сейчас кнопка "Поделиться" открывает редактор истории, но фон в Telegram остается черным. Это главный баг следующего обновления.

Что проверить:
- открыть `/debug/share-latest` после шаринга и убедиться, что `mediaUrl` ведет на свежую карточку;
- открыть `/debug/share-latest-image` с телефона и проверить, что картинка реально отображается вне Mini App;
- проверить заголовки ответа для `/share/*.jpg`: `Content-Type: image/jpeg`, HTTPS без редиректов, публичная доступность для Telegram;
- урезать `text` в `shareToStory` до 200 символов для обычных пользователей;
- попробовать PNG вместо JPG и сравнить поведение;
- если Telegram все равно показывает черный фон, сделать fallback через `shareMessage` / prepared inline message или обычный `t.me/share/url` с красивой ссылкой.

Официально `shareToStory(media_url, params)` требует HTTPS URL картинки/видео; значит, проблема почти точно в том, как Telegram получает или интерпретирует нашу картинку.

## P1: эволюция скинов по дороговизне

Идея: первые ракеты остаются компактными и простыми, дорогие получают более дорогой силуэт, но хитбокс не меняется. Для первых страниц механика остается честной; для поздних дорогих скинов можно дать очень маленький пассивный бонус, чтобы покупка ощущалась сильнее, но не превращалась в pay-to-win.

Предложение по силуэтам:
- VT / TON: базовый чистый корпус, один двигатель, аккуратные плавники.
- XRP / TRX / ADA: шире cockpit, ярче контур, маленький график-молния на боку.
- AVAX / DOT / BNB: двойной форсаж, боковые стабилизаторы, больше свечения.
- DOGE / PEPE / LINK: редкие игровые формы с характером, но без мемного перебора.
- SOL / ETH / BTC: премиальные корпуса, кольцевой двигатель, корона/призма/гравировка, более глубокий neon metal.

Пассивы, если добавлять:
- дешевые скины: без бонуса;
- средние: +2-3% радиус магнита или +2% длительность power-up;
- дорогие: +4-5% длительность power-up или +1 дополнительный VTC с супер-монетки;
- BTC/ETH: не менять хитбокс, только легкий престижный бонус и уникальный трейл.

## P2: визуальные рендеры

Сделать 2 набора рендеров:
- gameplay sprites: маленькие side-view ракеты, читаемые в 9:16 canvas;
- promo renders: крупные красивые версии для магазина, истории и профиля бота.

Важно: игровые спрайты должны иметь общий визуальный язык, но не быть копией одной формы с заменой текста.
